Images

Images can be inserted in an HTML document by using <img> tag. This does not create a copy of image. Instead, it only creates a reference to original image placed somewhere in the computer's storage. The <img> tag is an empty tag which means that it does not require a closing </img> tag. It contains ve attributes out of which two are required and three are optional.

Example

<img src="bridge.jpg" alt="Bridge" width="600">
