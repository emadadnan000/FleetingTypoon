HTML LIST

Lists are very useful in displaying point by point information such as to-do list, list of ingredients for a recipe, list of categories, etc. HTML provides three different types of list elements namely Unordered Lists, Ordered Lists and Description Lists. The web browsers automatically add indents, spaces and markers to HTML lists.

Ordered Lists

An ordered list also displays a list of related items. It is used where the order of the list is important e.g. names of students in order of their exam ranks. It is dened by <ol>; tag and each of its list item is dened by <li>. The web browser will display these list items with numbers starting from 1 instead of bullet points.

Unordered Lists

An unordered list is used to display a list of related items. It is defined by <ul> tag and each of its list item is defined by <li>. The web browser will display these list items with bullet points.

Description Lists

A description list is used to display a list of related items. It is defined by <dl> tag and each of its list item is defined by <dt> and <dd>. The web browser will display these list items with bullet points.
