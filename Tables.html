Tables

Tables allow displaying the content (like text, image, links) in the form of rows and columns. The coordinating place of a row with a column is called a cell. These cells contain some content of the webpage. In HTML, a table is defined by <table> tag.

Rows and Columns

A row is the collection of all horizontal cells of a table. A table can contain any number of rows. All the rows in a table have an equal number of cells. It is defined by <tr> tag which is placed inside the <table> tag. A column is the collection of all vertical cells of a table. A table can contain any number of columns as well. It is dened by <td> tag which is placed inside the <tr>row tag.

Table Heading

HTML table allows dening a header for the columns of our table. A header cell is dened by <td> tag and is placed inside a <tr> row tag. It is used to differentiate from the content placed inside normal <td> data cells. To make the entire header row stand out from rest of the table contents, defined <th> tags for all columns of the first <tr> row tag of any <table> tag.
